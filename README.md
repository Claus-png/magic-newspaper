# Magic Newspaper

Drag-and-drop конструктор внутриигровых газет для **Foundry VTT v13**.  
Мастер верстает — игроки читают в реальном времени.

---

## Возможности

- **9 типов элементов** — шапка, заголовок, текст, изображение, разделитель, цитата, блок, объявление, подпись
- **16 текстур бумаги** — Classic, Gothic, Cyber, Luxury, Noir, Vintage, Art Deco, Tech, Terminal, Grunge, Academic, Archive, Minimal, Industrial, Rustic, Construct
- **Галерея шаблонов** — 6 готовых стартовых макетов (вестник, плакат розыска, секретный документ, киберотчёт, королевский указ, срочный выпуск)
- **Экстренный выпуск** — оверлей поверх интерфейса всех игроков с анимацией и звуком
- **Real-time доставка** — публикация мгновенно отображается у всех клиентов через socket
- **Reveal-механика** — элементы раскрываются по клику игрока
- **Независимая навигация** — каждый игрок листает страницы в своём темпе
- **Кнопка «Показать всем»** — GM принудительно переключает страницу у всех
- **История действий** — Ctrl+Z / Ctrl+Y, до 50 шагов
- **Drag & Drop** — актёры, картинки и записи журнала прямо на холст
- **Автосохранение** — с кнопкой восстановления последней версии
- **Архив и черновики** — управление выпусками
- **Экспорт** — самодостаточный HTML-файл, JSON-бэкап
- **Сохранение в Журнал** — страница газеты попадает в личный журнал игрока
- **Права по ролям** — Create / Edit / Publish / View настраиваются независимо

---

## Использование

После включения модуля кнопки появляются в двух местах:
- **Боковая панель Notes** (иконка карты) — для GM и игроков
- **Журнал (Journal)** — кнопки «Редактор» и «Газета» в шапке списка

### Быстрый старт

1. Нажмите **Редактор** → откроется конструктор
2. Выберите шаблон через кнопку **Шаблоны** в тулбаре
3. Заполните текст и изображения
4. Нажмите **Выпустить** — игроки получат уведомление в чат

### Горячие клавиши редактора

| Клавиша | Действие |
|---------|----------|
| Ctrl+Z / Ctrl+Y | Отмена / Повтор |
| Ctrl+S | Сохранить |
| Ctrl+D | Дублировать выбранный элемент |
| Стрелки | Точное позиционирование (Shift = по сетке) |
| F | Подогнать зум под окно |
| G | Переключить сетку |
| Escape | Снять выделение |
| Delete / Backspace | Удалить элемент |

## Известные ограничения

- Изображения, вставленные через drag-and-drop и автокроп, сохраняются как base64 внутри настроек мира — крупные портреты могут увеличить размер данных
- Google Fonts требуют интернет-соединения; при отсутствии сети шрифты заменятся системными
